Základní příkazy Gitu

Vytvoření lokálního repozitáře


git init Vytvoření lokálního repozitáře

git --version Příkaz vypíše aktuální verzi gitu

git config --local user.user "uzivatel" Nastavení uživatelského jména pro celý počítač (--local)

git config --local user.email "E-mail" Nastavení E-mailu pro celý počítač (--local)

Zaměněním --local za --global platí nastavení pro konkrétní repozitář

Příkazy pro lokální repozitář


git status Zjištění stavu repozitáře

Červeně označené soubory nejsou součástí staging area

Zeleně označené soubory jsou v staging area

git add -all / git add . Umístění všech souborů do staging area

git add "název_souboru" Umístění konkrétního souboru do staging area

git reset Uvolní se všechny soubory ze staging area

git reset "název_souboru" Uvolní se konkrétní soubor ze staging area

Poropojení vzdáleného a lokálního repozitáře


git remote add origin "odkaz" Připojí vzdálený repozitář pod názvem origin (obvykle zadané pojmenování), aby mohl být synchronizován s lokálním repozitářem.

git push -u origin master Aktualizuje stav vzdáleného repozitáře (origin) podle lokálního repozitáře (master - základní, hlavní větev repozitáře).

git pull origin master Aktualizuje stav lokálního repozitáře podle vzdáleného repozitáře

git log Příkaz vypíše přehled commitů

git clone "odkaz" Vytvoření klonu vzdáleného repozitáře na lokálním počítači.
